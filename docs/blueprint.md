# **App Name**: ScriptureStream

## Core Features:

- Multi-Language Bible Browser: Navigate by book, chapter, and verse across supported Bible versions (e.g., KJV, NIV) and display content in Bisaya, Tagalog, or English, with quick language/version switching.
- Presentation Mode Display: Activate a simplified, full-screen display of the current verse or passage with large, readable typography, optimized for public viewing or projected screens.
- Second Screen Output: Utilize browser APIs to open Presentation Mode content in a new, fullscreen browser window, enabling display on an extended monitor (HDMI output emulation).
- Local Bible Content Database: Store and retrieve all supported Bible texts (Bisaya, Tagalog, English) using a local database (SQLite, JSON, or MongoDB) to ensure offline access and fast retrieval.
- Contextual Verse Finder Tool: An AI-powered tool that suggests relevant Bible verses based on user-entered keywords or a short descriptive phrase, offering spiritual insights.
- Quick Keyword Search: Enable users to quickly search for specific words or phrases within the currently active Bible version and language.

## Style Guidelines:

- Color scheme is light, prioritizing clarity and readability. The primary color is a muted indigo blue (#4D4073), conveying depth and contemplation. The background is a very light, desaturated hint of this hue (#FAF9FC), offering a clean canvas. An accent color, a vibrant sky blue (#2EB3F7), is used for highlights and interactive elements.
- For scripture text and presentation display, 'Alegreya' (humanist serif) is recommended for its elegant, intellectual, and contemporary feel, suitable for extended reading and large screens. For UI elements and navigation, 'Inter' (grotesque-style sans-serif) provides a modern, neutral, and highly legible experience.
- Icons should be minimalistic, clean, and utilize line-based designs to maintain a sophisticated and spiritual aesthetic, complementing the overall design without distraction. Prioritize widely recognized symbols for navigation and settings.
- The layout is responsive and adaptive, featuring a clean content pane for Bible text. Navigation controls are easily accessible, optimizing for both desktop and mobile viewing. Presentation mode offers a distraction-free, full-screen display with centralized text.
- Subtle and smooth fade-in/fade-out transitions are used for changing verses or passages to provide a polished user experience. Animations for UI interactions are kept minimal and performant.
